jjproxy

轻量级的翻墙工具，不需要安装客户端。可以设置系统代理，也可以设置浏览器代理或者配合插件使用。

PAC地址

https://liruqi.github.io/jjproxy/pub.pac

Linode上搭建的公共代理，速度可能较慢，不保证可用性。

可以发邮件至 liruqi@gmail.com 购买高速代理，64人民币一年；如果你会用 shadowsocks 客户端，只需要一个购买 shadowsocks 账号，6.4人民币一年。

使用方法

先介绍一下设置各个系统代理的方法。

Windows

Internet选项 -> 连接 选项卡 -> 局域网设置(如果是电脑拨号上网, 这里点'设置') -> 使用自动配置脚本 -> 填入PAC地址 -> 确定

Mac OS X

系统设置 -> 网络 -> 高级 -> 代理 -> 自动代理配置 -> URL中填入PAC地址 -> 好

iOS (iPhone/iPad)

  1. 打开设置, 选择Wi-Fi (图)
  2. 选择当前使用的热点 (图)
  3. 拖到最下面的代理设置，选择"自动"，填写PAC地址 (图)
